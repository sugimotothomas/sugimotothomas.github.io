Projects

Card Game Simulator

Developed in late 2020 this is a web app built using node.js and the HTML5 framework Phaser 3 for Oregon State's Computer Science Group Capstone Project. The app lets anyone join around a table and play any game of cards they would like. Since it is a simulator, it's up to you to enforce the rules! The idea was conceived by me and I was in charge of leading our group. Includes a dynamic table and hand system, node.js server, and build from the ground up by me.

smallsh.c

A bash-like shell application written in C for my Operating Systems Class. This application lets users use commands to navigate directories (such as cd, status, and exit), execute other programs with execvp both in the application's foreground or background, and makes custom signal handlers for SIGINT and SIGTSTP.
